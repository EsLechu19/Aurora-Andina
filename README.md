# Aurora Andina - Landing Page

Landing page estática para **Aurora Andina**, una propuesta visual de turismo premium enfocada en viajes privados, experiencias culturales y rutas boutique por Sudamérica.

## Descripción

El proyecto presenta una página de una sola vista con navegación por secciones, diseño responsive, menú móvil, animaciones de aparición al hacer scroll e imágenes externas cargadas desde Unsplash.

## Estructura

```text
pagina-copiada/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## Tecnologías

- HTML5
- CSS3
- JavaScript vanilla
- Google Fonts
- Imágenes remotas de Unsplash

## Secciones Principales

- **Inicio:** Hero principal con mensaje de marca y llamadas a la acción.
- **Experiencia:** Paneles visuales sobre alojamiento, paisaje y cultura.
- **Rutas:** Listado de itinerarios privados sugeridos.
- **Galería:** Imágenes destacadas de atmósferas de viaje.
- **Contacto:** Información de atención y enlace de correo.

## Cómo Ejecutarlo

No requiere instalación de dependencias.

1. Abre la carpeta `pagina-copiada`.
2. Ejecuta el archivo `index.html` directamente en el navegador.

También puedes servirlo con una extensión como **Live Server** en Visual Studio Code para trabajar con recarga automática.

## Personalización

- Cambia los textos principales en `index.html`.
- Ajusta colores, tipografías, espaciados y estilos responsive en `css/style.css`.
- Modifica el comportamiento del menú móvil y las animaciones de aparición en `js/main.js`.
- Reemplaza las URLs de imágenes por recursos propios si necesitas una versión final de producción.

## Notas

Este proyecto es una landing page estática. No incluye backend, formularios funcionales ni proceso de build.
